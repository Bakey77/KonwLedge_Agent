  ┌──────────────────────┬───────────────┬───────────────────────────────┬─────────────────────────┬────────────────┬─────────────────┐
  │         步骤         │  file_paths   │            chunks             │       extractions       │ vectors_stored │ entities_stored │
  ├──────────────────────┼───────────────┼───────────────────────────────┼─────────────────────────┼────────────────┼─────────────────┤
  │ parse 完成后         │ ["paper.pdf"] │ [Chunk0, Chunk1, ..., Chunk9] │ []                      │ 0              │ 0               │
  ├──────────────────────┼───────────────┼───────────────────────────────┼─────────────────────────┼────────────────┼─────────────────┤
  │ extract 完成后       │ 同上          │ 同上                          │ [Result0, ..., Result9] │ 0              │ 0               │
  ├──────────────────────┼───────────────┼───────────────────────────────┼─────────────────────────┼────────────────┼─────────────────┤
  │ store_vectors 完成后 │ 同上          │ 同上                          │ 同上                    │ 10             │ 0               │
  ├──────────────────────┼───────────────┼───────────────────────────────┼─────────────────────────┼────────────────┼─────────────────┤
  │ store_graph 完成后   │ 同上          │ 同上                          │ 同上                    │ 10             │ 156             │
  └──────────────────────┴───────────────┴───────────────────────────────┴─────────────────────────┴────────────────┴─────────────────┘

  最终 vectors_stored=10 表示10个文本块已可被检索，entities_stored=156 表示从论文中抽取了156个实体存入知识图谱。
